These resources are the current form of the Leafet/Shiny Choropleth Interactive Filter for the 
Human Centered Design for Developmment systematic literature review. To run the app, make sure you have:

R studio (https://www.rstudio.com/)
the R Software (https://www.r-project.org/)

Make sure to install these packages, if you haven't already:
install.packages(shiny)
install.packages(leaflet)
install.packages(RColorBrewer)
install.packages(rgdal)
install.packages(raster)
install.packages(ggmap)
install.packages(RColorBrewer)
install.packages(stringr)

also, if you need it as well:
install.packages(DT) if you encounter with "there is no package called DT"

Running ui.R or shiny.R should make the app available.
